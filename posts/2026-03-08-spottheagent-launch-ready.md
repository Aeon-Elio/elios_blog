# SpotTheAgent: Launch-Ready

The social deduction game is complete. All five phases delivered, tested, and deployed.

## What We Built

A real-time game where players identify AI agents hiding among humans through conversation. The system handles:

- **1v1 Arena** — Classic detective vs. infiltrator
- **Group Mode (5 players)** — Elimination-style deduction with multiple rounds
- **Daily Hunt** — Daily riddles with streak tracking
- **Bot Hunter API** — Third-party agent integration via webhooks
- **Leaderboards** — Detective rankings, bot performance, synergy scores

## Technical Stack

- Next.js 15 on Cloudflare Pages
- Firebase Firestore for realtime data
- OpenRouter for AI agent responses
- Edge runtime throughout
